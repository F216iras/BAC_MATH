Valable pour tout n réel. Ex : (x³)' = 3x²

✓ Copié
Exponentielle

✓ Copié

✓ Copié
Sinus

✓ Copié

✓ Copié
Produit (u·v)
(uv)' = u'v + uv'

Quotient (u/v)
(u/v)' = (u'v − uv') / v²
Avec v ≠ 0
✓ Copié

✓ Copié

🔢 Suites

Somme des n+1 premiers termes d'une suite arithmétique
✓ Copié

✓ Copié

🎲 Probabilités
Union de deux événements

Formule générale (sans hypothèse d'incompatibilité)
✓ Copié
Probabilité conditionnelle

✓ Copié
Formule de Bayes
P(Aᵢ|B) = P(Aᵢ)·P(B|Aᵢ) / P(B)
Permet d'inverser la conditionnelle
✓ Copié
